# EEZ Studio 项目 — 智趣宝盒 Home 页面设计

## 环境要求

- [EEZ Studio](https://github.com/eez-open/studio/releases) 最新版（支持 LVGL 9.x）

## 新建项目参数

| 参数 | 值 |
|------|-----|
| 屏幕宽度 | 1024 px |
| 屏幕高度 | 600 px |
| 色深 | RGB888 (24-bit) |
| LVGL 版本 | 9.x |
| 项目名称 | smart-learning-box |

```
┌──────────────────────────────────────────┐

├──────────────────────────────────────────┤
│                                          │
│   ┌──────────────┐  ┌──────────────┐     │
│   │   AI 老师    │  │  拍照解题     │     │  ← 卡片 440×220
│   │  智能问答辅导 │  │  拍题秒出答案 │     │     圆角 16px
│   └──────────────┘  └──────────────┘     │     间距 24px

│   ┌──────────────┐  ┌──────────────┐     │
│   │  数学 2048   │  │  成就系统     │     │
```

## 导出步骤

1. EEZ Studio 菜单 → **Build** → **Export LVGL C Code**
2. 导出目标选择 `assets/eez/exported/`
3. 导出后目录结构应为：

```
assets/eez/exported/
├── ui.c          ← 主 UI 文件
├── ui.h          ← 声明
├── fonts/        ← 字体 C 文件
└── images/       ← 图片 C 文件
```

4. 导出完成后告诉我，我来把代码集成到 `main/app/home/ui/` 并接入 `home.c`
