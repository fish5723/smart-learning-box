# 🗺 Roadmap

## ✅ Phase 1 - Core Platform (已完成)

- [x] ESP32-P4 系统移植与 BSP 驱动（LCD/Touch/Camera/Storage/WiFi）
- [x] LVGL 9.x UI 框架集成，暗色主题设计系统
- [x] 双任务架构（UI 任务 + 业务任务）
- [x] 首页、启动页、WiFi 配置页

## 🚧 Phase 2 - AI 与 OCR（已完成）

- [x] Vision Core 状态机框架（6 阶段流水线）
- [x] OCR 拍照搜题全链路（OV5640 → Qwen3-VL → 豆包大模型）
- [x] AI 对话模块（SSE 流式响应）
- [x] 答案缓存系统（MD5 哈希去重）
- [x] 错题本自动归档

## 🚧 Phase 3 - 游戏化（已完成）

- [x] NES 模拟器移植（InfoNES 内核，20-50fps）
- [x] 触摸虚拟手柄
- [x] TF 卡 ROM 加载 + 存档管理
- [x] 游戏中心（2048 / 数学闯关 / 单词王）
- [x] 成就系统（积分/等级/徽章）

## 📋 Phase 4 - 优化与完善（进行中）

- [ ] NES 模拟器帧率优化（目标 50-60fps，当前瓶颈分析）
- [ ] PSRAM 使用率优化（当前 25.7%，降低到 20% 以下）
- [ ] WiFi 连接稳定性增强（SDIO 重连机制优化）
- [ ] ESP32-C6 协处理器 OTA 固件升级

## 🔮 Phase 5 - 新功能规划

| 功能 | 说明 | 优先级 |
|------|------|--------|
| **语音交互** | 集成 ASR（语音转文字），支持语音提问 | ⭐⭐⭐ |
| **拍照翻译** | OCR 识别后直接翻译，外语学习场景 | ⭐⭐⭐ |
| **学习报告** | 按周/月生成学习数据统计报告 | ⭐⭐ |
| **AI 错题讲解** | 错题自动推送 AI 知识点讲解视频/语音 | ⭐⭐ |
| **家长控制** | 使用时长限制、内容过滤、学习进度推送 | ⭐⭐ |
| **多语言界面** | 英文/日文界面支持 | ⭐ |
| **蓝牙手柄** | 支持外接蓝牙游戏手柄 | ⭐ |

## 💡 远期构想

- **2.0 版本**：升级到 ESP32-P5（如可用），提升 NES 模拟器到 60fps 稳定，支持 SNES/GB 模拟
- **Cloud Sync**：学习数据云端同步，支持手机端查看学习报告
- **App Store**：开放式游戏/应用商店，允许社区上传自定义内容
- **教育内容合作**：对接学校教材标准，提供课标同步练习题
